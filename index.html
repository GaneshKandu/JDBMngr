JDBMngr

         Java Based web Application to manage your DataBase ( DB Tool ) JDBGmgr is a Java Based Web Application to access your DataBase to perform your daily operation

View on GitHub Download .zip Download .tar.gz

Features

  • Creating a database
  • Connection to an existing database
  • Upload a sql file
  • Import a sql file
  • Managing multiple databases
  • Inserting data
  • Modifying data
  • Manual creation
  • Sql Query box

Installation

  • Take the JDBMngr.war file and deploy it, by copying it to $TOMCAT_HOME/webapps.
  • Restart Tomcat and point your browser to http://YOURIP:YOURPORT/JDBMngr/.

or

  • Open your TOMCAT Manager and Select JDBMngr.war and Diploy

or

  • Copy all the file and folder to your $TOMCAT_HOME/webapps/JDBMngr folder
  • css
  • Images
  • js
  • META-INF
  • WEB-INF
  • index.jsp
